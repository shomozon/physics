4.連星

\begin{eqnarray} m\boldsymbol a = -G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \\ M\boldsymbol A = G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \\ m\boldsymbol a = -G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \Leftrightarrow m(\boldsymbol a_{B→A} + \boldsymbol A) = -G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \\ \Leftrightarrow m\boldsymbol a_{B→A} = -G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} - \dfrac{m}{M}G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \\ \Leftrightarrow m\boldsymbol a_{B→A} = -(1 + \dfrac{m}{M})G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \end{eqnarray} Bに対してAはBを焦点とする楕円軌道を描く。

\( M\boldsymbol A = G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \Leftrightarrow M\boldsymbol a_{A→B} = -(1 + \dfrac{M}{m})G\dfrac{mM}{r_{A→B}^3}\boldsymbol r_{A→B} \)

Aに対してBはAを焦点とする楕円軌道を描く。
Bに対してAが最近点にいる時、速さが\( \ v_{0} \ \)、距離が\( r_{0} \)の場合、 \( m\boldsymbol a_{B→A} = -(1 + \dfrac{m}{M})G\dfrac{mM}{r_{B→A}^3}\boldsymbol r_{B→A} \Leftrightarrow r_{B→A} = \dfrac{\dfrac{h^2}{G(m + M)}}{1 + \dfrac{h^2A}{G(m + M)}\cos (\theta - \theta_0)} \)

\( \theta_0 = 0 \ \)として、\( h = r_{0}v_{0}, \ \theta = 0 \ の時、r_{B→A} = r_{0} \)より、 \( A = \dfrac{1}{r_{0}} - \dfrac{G(m + M)}{h^2} \)

\( r_{B→A} = \dfrac{\dfrac{h^2}{G(m + M)}}{1 + (\frac{h^2}{r_{0}G(m + M)} - 1)\cos \theta} \)

\( r_{A→B} = \dfrac{\dfrac{h^2}{G(m + M)}}{1 + (\frac{h^2}{r_{0}G(m + M)} - 1)\cos \theta} \)
